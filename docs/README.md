# Genesis Documentation

Este repositorio contiene la documentación técnica y operativa del proyecto **Genesis** (Spring Boot + PostgreSQL + Jenkins CI/CD).

## Contenido
- 00-overview: alcance, endpoints, objetivos
- 01-architecture: diagramas y componentes
- 02-ci-cd: Jenkins pipeline, credenciales, labels
- 03-envs: BUILD/TEST/PROD
- 04-deploy: systemd, releases, rollback
- 05-ops-runbooks: operación diaria
- 06-troubleshooting: fallas típicas
- 07-security: llaves, permisos, hardening
- 08-splunk: logs y forwarder
